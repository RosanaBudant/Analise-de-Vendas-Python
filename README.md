# Análise de Dados de Vendas com Python

Projeto simples de análise de dados utilizando Python.

## Tecnologias
- Python
- Pandas
- Matplotlib

## Objetivo
Explorar um dataset de vendas e gerar insights como:

- faturamento total
- produtos mais vendidos
- faturamento por categoria

## Resultados
O script gera análises no terminal e um gráfico de faturamento por categoria.
